# Proctoring System

## Overview
The Proctoring System is a web application designed to facilitate online examinations and monitoring for students, proctors (teachers), and administrators in an engineering college. This system provides three distinct dashboards tailored to the needs of each user role.

## Features
- **Student Dashboard**: A dedicated interface for students to access their exams, view results, and manage their profiles.
- **Proctor Dashboard**: A specialized dashboard for proctors to monitor exams, manage student activities, and ensure academic integrity.
- **Admin Dashboard**: An administrative interface for managing users, overseeing the examination process, and configuring system settings.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in a web browser to access the application.
